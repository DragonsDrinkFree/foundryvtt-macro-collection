# foundryvtt-macro-collection
A simple Collection of foundry macros I've found useful.

These macros are provided as simple .js files, but can be used by copy/pasting them into a FoundryVTT Script Macro.


# Included Macros


## SetWeatherEffects
This is a quick Quality of Life macro for Scene controls. It brings up a list of scene weather effects for default Foundry scene's.

Normally these must be set in the Ambiance tab of the scene configuration. This macro will set those values directly with with an easy UI.


## TokenAttributeBrowser
Select a token on the Scene Canvas and trigger macro. Brings up a window that allows for searching/filtering of actor attributes.
This is helpful for writing other macros, or for setting up modules that are system independent and require specific system knowledge to be configured.


## TokenDispositionChanger
Select 1 or more tokens. The macro allows you to quickly set their disposition: Hostile, Friendly, Neutral, Secret.
